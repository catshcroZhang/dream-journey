# 寻梦之旅 Skill

> "梦里寻踪，现实成真"

## 简介

「寻梦之旅」是 Fly.ai 官方 Skill，帮助用户把反复出现在梦里的模糊场景，主动"寻"成真实可去的旅行。让用户在现实中找到、验证并重返那个梦境，产生强烈宿命感和治愈情绪。

## 核心能力

1. **梦境高清还原** - 将模糊的梦境描述扩展成电影级场景描述
2. **现实寻梦匹配** - 调用 Fly.ai 实时匹配高度相似的真实目的地
3. **寻梦行程规划** - 端到端行程规划，支持一键预订
4. **行中梦境验证** - 拍照讲解，对比"梦境 vs 现实"
5. **情感化输出** - 生成《梦成真了》情感报告和短视频

## 前置依赖

### 必需工具

1. **FlyAI CLI**
   ```bash
   npm i -g @fly-ai/flyai-cli
   ```

2. **Node.js**（v14 或更高版本）- 用于运行辅助脚本
   - 下载地址：https://nodejs.org/
   - 安装后验证：`node --version`
   - npm 会随 Node.js 一起安装，验证：`npm --version`

> 💡 **提示**：如果你只需要 AI 对话功能（不使用辅助脚本），只需安装 FlyAI CLI 即可。
> 但如果想生成 HTML 报告、短视频脚本等增强内容，必须安装 Node.js。

## 使用方式

### 基础流程

1. 描述你的梦境（无论多模糊都可以）
2. AI 会高清还原你的梦境
3. 匹配 2-3 个真实目的地选项
4. 选择你心仪的目的地
5. AI 生成完整行程并支持预订
6. 行程中拍照验证梦境
7. 行程结束生成情感报告和短视频

### 增强工具（可选）

本 Skill 包含 4 个辅助脚本，位于 `scripts/` 目录：

```bash
# 1. 生成梦境视觉 Prompt（用于 AI 绘画）
node scripts/generate-dream-prompt.js "你的梦境描述"

# 2. 生成星空主题 HTML 报告
node scripts/generate-report.js --json data.json

# 3. 生成 10 个短视频脚本（抖音/小红书）
node scripts/generate-video-scripts.js --dream "梦境" --dest "目的地" --score 92

# 4. 生成结尾金句（5大主题50条）
node scripts/generate-quote.js --dream "梦境描述"
```

**详细文档**：
- 脚本使用说明：`references/README.md`
- 图片添加指南：`references/HOW-TO-ADD-PHOTOS.md`
